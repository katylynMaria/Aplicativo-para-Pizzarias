# Aplicativo para Pizzaria

Este projeto é um **aplicativo web** desenvolvido com **HTML, CSS e JavaScript**, criado para simular o sistema de pedidos de uma pizzaria.  
A aplicação permite visualizar produtos, montar pedidos e calcular automaticamente os valores, oferecendo uma interface simples, intuitiva e responsiva.

---

## Objetivo

O objetivo do projeto é aplicar os conhecimentos de **desenvolvimento front-end** e **design de interfaces**, criando um sistema prático e funcional voltado ao atendimento de uma pizzaria.  
O foco principal está na organização do layout, usabilidade e implementação de lógicas básicas de interação com o usuário por meio de JavaScript.

---

## Funcionalidades

- Exibição de produtos disponíveis (pizzas, bebidas, acompanhamentos etc.);
- Cálculo automático do valor total do pedido;
- Interface moderna e responsiva;
- Organização dos itens selecionados;
- Simulação do processo de finalização do pedido.

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas;
- **CSS3** – Estilização e responsividade;
- **JavaScript** – Lógica e interatividade.
